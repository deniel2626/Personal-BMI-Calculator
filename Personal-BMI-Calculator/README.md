# 💪 Personal BMI Calculator

A responsive web application built with Python and Streamlit that calculates Body Mass Index (BMI) and provides personalized health recommendations.

## Features

- BMI Calculation
- Weight Category Classification
- Target Weight Recommendations
- Weight Gain Suggestions
- Weight Loss Guidance
- Calorie Burn Estimates
- Step Count Estimates
- Dark Mode / Light Mode
- Mobile-Friendly Interface

## Built With

- Python
- Streamlit
